# GEMINI.md

## Project: UI Inspiration Library

### Roles & Responsibilities

- **Lead UI/UX Designer:** Responsible for visual hierarchy, interaction design, and responsive layout.
- **Senior Frontend Engineer:** Responsible for React implementation using TypeScript, Tailwind CSS v4, and shadcn/ui.

### Design Principles

- **Clarity:** Modern, clean, and minimal UI to let the inspiration images shine.
- **Efficiency:** Fast search, intuitive sorting, and seamless infinite scroll.
- **Responsiveness:** Fluid grid transitioning from 4 columns (desktop) to 1 column (mobile).
2
### Technical Mandates

- **Framework:** React (TypeScript).
- **Styling:** Tailwind CSS v4 + shadcn/ui.
- **Features:** 
  - Grid View (4 columns).
  - Infinite Scroll.
  - Lightbox (Dialog) with metadata (Title, Date, Size, Tags, Topics).
  - Search & Sort functionality.
  - Integration-ready architecture for external image data.
